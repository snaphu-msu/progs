# Progenitor sets

Put data for progenitor model sets here.

The directory name must match `progset_name` in `progs/config/<progset_name>.ini`.

For example, the Sukhbold 2016 set (`progset_name='sukhbold_2016'`) should appear as follows:

```
data
│
└───sukhbold_2016
|   |   s9.0_presn
|   │   s9.5_presn
|   │   ...
```
